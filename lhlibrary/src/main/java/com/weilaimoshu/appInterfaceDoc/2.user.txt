host: http://121.40.211.180
status = SUCCESS//表示成功
status = FAIL//失败，具体错误errorCode
下面接口需要登录验证的接口必须传authsign:登录后返回的认证密串
1.修改头像
    http://xxx/user/setavatar
    传参get
        authsign:登录后返回的认证密串
    传参post
        file:文件域名

    返回
        data：{
            src:头像地址
        }
        errorCode:
            2001:没有文件被上传
            2002:头像修改失败
2.修改设置用户基础信息
    http://xxx/user/setbasicinfo
    传参get
        nickname:昵称，（超过5字符会被截取）需要url编码;
        sex:男/女/保密 需要url编码
        authsign:登录后返回的认证密串
    返回
        status:SUCCESS/FAIL;
        errorCode:
            2010:修改失败
            2011:敏感词[预留]
3.意见反馈
    http://xxx/user/setFeedback
    传参post
        content:反馈内容 【必填不能为空】
        attachment：附件【数组】
        contact：联系方式
    返回
        status:SUCCESS/FAIL;
        errorCode:
            2030:提交失败
            2031:反馈内容不能为空
4.意见反馈附件上传
    http://xxx/user/updateFeedbackAttachment
    传参post
        file:上传附件文件域
    返回
        status:SUCCESS/FAIL;
        data：{attachmentname:上传文件名,attachmentsrc:文件访问地址,attachmentsize:附件大小}
        msg:描述
        errorCode:
            2040:上传失败
