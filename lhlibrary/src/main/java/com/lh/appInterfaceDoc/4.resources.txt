1.资源分类

2.资源列表
    http://xxx/resources/lists
    传参GET
        page:页数
        resources_type_id:分类id 【可空，不指定分类】
        resources_attr:资源属性，1表示原创，2表示搬运【可空。空表示不指定】
        sort:hot热门作品,late最新作品【可空。空默认按时间】
    返回
        data：[{
            resources_id:资源id 
            resources_type_id:资源类型id
            title：标题
            sub_title：副标题
            head_img：封面图片
            describe：描述
            come_from：1表示小编发布，2表示p主投稿
            resources_status：1表示待发布，2表示已经发布、3表示已屏蔽
            collection_totle：收藏次数
            comment_totle：评论数
            praise_totle：点赞数
            add_time：资源创建时间
            uid：投稿人id
            author：作者
            resources_attr：资源属性，1表示原创，2表示搬运
            resources_idol_id：偶像id
            tags：标签
            online：1:线上2线下
            audit_status：1:待审核2审核通过3审核未通过
            uuid:uuid
            attachment_src:附件地址
            timelong:时长
            imgs:列表显示多图片
            version:游戏版本号
            chapternum:章节数
            info_url:详情直接html5地址，注意这里具体看手机端和H5如何操作，用不用淡定【预留字段】
        },.....]
3.资源详情[针对已发布]
    http://xxx/resources/info
    传参GET
        uuid:资源uuid
        authsign:[可空]传了登陆之后的authsign值可以用于判断搜藏状态
    返回
        data：{
            resources_id:资源id
            resources_type_id:资源类型id
            title：标题
            sub_title：副标题
            head_img：封面图片
            content：内容
            describe：描述
            come_from：1表示小编发布，2表示p主投稿
            resources_status：1表示待发布，2表示已经发布、3表示已屏蔽
            collection_totle：收藏次数
            comment_totle：评论数
            praise_totle：点赞数
            add_time：资源创建时间
            uid：投稿人id
            resources_attr：资源属性，1表示原创（作者），2表示搬运（投稿人）
            resources_idol_id：偶像id
            resources_idol_name:关联的偶像[{idol_id:偶像id,name:偶像名},{}]
            tags：标签
            online：1:线上2线下
            audit_status：1:待审核2审核通过3审核未通过
            uuid:uuid
            online_time：上线时间
            type_name：类型名
            method：类型标示
            chapter：0部分章节1分章节显示
            chapternum:章节数
            isFavorite:1已收藏0未收藏
            isPraise 1已点赞，0未点赞
        }
4.资源评论
    http://xxx/resources/getComment
    传参GET
        num:每页取条数，默认10条,非特殊情况不做修改
        resources_id:资源id
        page:页数
        sort:默认评论时间降序,hot:点赞数降序
    返回
        data：{count:总评论数,lists:[{status:1评论正常2评论被屏蔽(直接展示：该条评论已被屏蔽!),isPraise 1已点赞，0未点赞},....]}
5.1.章节列表
    http://xxx/resources/getChapterList
    传参GET
        uuid:资源uuid
    返回
        data：[{
            chapter:章节号

        }]
5.2.章节详情
    http://xxx/resources/getChapterInfo
    传参GET
        uuid:资源uuid
        chapter:章节号
    返回
        data：{
            resources_id:资源id
            chapter:章节数
            title:章节标题
            head_img:封面图片
            imgs：漫画内容（多个图片的数组,漫画时候返回内容）
            content:小说内容 (小说时候返回内容，漫画时候空字符串)
            add_time：添加时间
        }
6.资源分享回调（用于统计,暂定）

         resources_id:资源resources_id
